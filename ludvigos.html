LUR - Ludvig Universal Repository

Мощный менеджер пакетов для настоящих гиков

LUR Менеджер Пакетов

Доступные команды:

$ lur install minecraft
Установить Minecraft (встроен в систему)
$ lur install superlauncher
Установить SuperLauncher
$ lur update
Обновить все пакеты
$ lur remove 32bit-legacy
Удалить 32-bit пакеты (рекомендуется)
LUR Terminal v1.2

$ lur search minecraft

Найден пакет: minecraft-kernel (v1.2)

$ lur install minecraft-kernel

МИНИМАЛЬНЫЕ:

  • 64-bit процессор (обязательно!)

  • 2 ГБ места на диске
