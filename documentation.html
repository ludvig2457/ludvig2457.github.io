Installing SuperLauncher

Step-by-step instructions for each operating system

Windows

  1. Download SuperLauncher-Win.exe.
  2. Run the installer and follow the instructions.
  3. Open SuperLauncher and log in with your Minecraft account.

Linux (.deb / .rpm)

  1. Download the corresponding package (.deb or .rpm).
  2. For .deb: sudo dpkg -i SuperLauncher.deb
  3. For .rpm: sudo rpm -i SuperLauncher.rpm
  4. Launch the launcher from the menu or terminal.

FAQ

Which systems are supported?

SuperLauncher supports Windows 10/11 (64-bit) and most Linux distributions (.deb & .rpm packages).

Can I install mods?

Yes! SuperLauncher supports easy mod management with installation and updates in one click.

Is it safe to use?

SuperLauncher is built with security in mind and protects your Minecraft accounts.
