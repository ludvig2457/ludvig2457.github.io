Installation Guide

Step 1: Download

Click the download button for your platform above to get the latest SuperLauncher installer.

Step 2: Install

Run the installer and follow the on-screen instructions. For Linux, use your package manager.

Step 3: Launch

Open SuperLauncher, log in with your Minecraft account, and start playing immediately.

FAQ

Is SuperLauncher safe to use?

Yes, it's built with security in mind to protect your Minecraft account.

Can I install mods?

Yes! SuperLauncher has built-in support for mods with easy installation.

Which platforms are supported?

Windows 10/11 and Linux (Debian/Ubuntu and RPM-based distros).
