Installation Guide

Windows Installation

  1. 1 Download the .exe file from the Windows download link
  2. 2 Run the installer as Administrator
  3. 3 Follow the installation wizard steps
  4. 4 Launch SuperLauncher from Start Menu or Desktop

Linux Installation

Ubuntu/Debian:

  1. 1 Download the .deb package
  2. 2 sudo dpkg -i SuperLauncher1.4.0.8.deb
  3. 3 Fix dependencies: sudo apt-get install -f

Arch Linux:

  1. 1 Download the .pkg.tar.zst package
  2. 2 sudo pacman -U superlauncher-1.4.0.8-1-x86_64.pkg.tar.zst
  3. 3 Launch from applications menu or terminal

Android Installation

  1. 1 Download the APK file from the Android download link
  2. 2 Enable "Install from unknown sources" in Settings
  3. 3 Open the downloaded APK file and install
  4. 4 Launch SuperLauncher from your app drawer

Frequently Asked Questions

Is SuperLauncher safe to use?

Yes! SuperLauncher is built with security as a top priority. We use industry-standard encryption to protect your Minecraft account credentials and ensure your gaming experience remains secure across all platforms.

Does Android version support mods?

The Android version currently focuses on Bedrock Edition with optimized performance for mobile devices. Full mod support is planned for future updates as we expand mobile functionality.

Which platforms are supported?

SuperLauncher now supports Windows 10/11 (64-bit), Linux (Debian/Ubuntu and Arch distributions), and Android 7.0+. We're continuously working to expand platform support and improve cross-platform compatibility.

How do I update SuperLauncher?

SuperLauncher includes automatic update functionality on all platforms. You'll be notified when new versions are available, and you can update with a single click. Manual updates can be done by downloading the latest version from our documentation.

Can I use the same account on different platforms?

Yes! Your Minecraft account works seamlessly across all SuperLauncher platforms. You can start playing on Windows, continue on Linux, or enjoy mobile gaming on Android with the same account.
